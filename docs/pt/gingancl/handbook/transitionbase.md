### Definição e uso { data-search-exclude }

O elemento [&lt;transitionBase&gt;](transitionbase.md) agrupa um conjunto de [&lt;transition&gt;](transition.md)
elementos, cada um dos quais define um efeito de transição em [&lt;media&gt;](media.md) elementos
na apresentação.

### Sintaxe { data-search-exclude }

| Elemento | Atributos | Conteúdo | Pai |
| --- | --- | --- | --- |
| [&lt;transitionBase&gt;](transitionbase.md) | *id* | ([&lt;importBase&gt;](importbase.md),[&lt;transition&gt;](transition.md))+ | [&lt;head&gt;](head.md) |

### Atributos { data-search-exclude }

| Nome | Valor | Descrição |
| --- | --- | --- |
| *id* | Pode receber qualquer valor de string que comece com uma letra ou sublinhado e que contenha apenas letras, dígitos, <"."> e <"_">. | O atributo *id* identifica exclusivamente o elemento dentro de um documento. |
