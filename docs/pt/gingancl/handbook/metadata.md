### Definição e uso { data-search-exclude }

O elemento [&lt;metadata&gt;](metadata.md) contém meta informações sobre um NCL
documento. Não é utilizado pelo reprodutor NCL para apresentação da aplicação,
mas pode ser usado para outros agentes de usuário NCL.

O elemento [&lt;metadata&gt;](metadata.md) atua como o elemento raiz de uma árvore RDF.

### Sintaxe { data-search-exclude }

| Elemento | Atributos | Conteúdo | Pai |
| --- | --- | --- | --- |
| [&lt;metadata&gt;](metadata.md) | -- | -- | [&lt;body&gt;](body.md) ou [&lt;context&gt;](context.md) |
