### Definição e uso { data-search-exclude }

A tag [&lt;head&gt;](head.md) é o contêiner para todos os elementos principais.

Elementos filhos do elemento [&lt;head&gt;](head.md) promovem instalações de reutilização em NCL
documentos ou definir meta-informações para agentes de usuário de aplicativos NCL.

O elemento [&lt;head&gt;](head.md) não possui nenhum atributo.
