### Definição e uso { data-search-exclude }

A tag [&lt;body&gt;](body.md) define o corpo do documento.

Todos os ativos de mídia que definem uma aplicação NCL juntamente com relacionamentos
entre esses ativos estão definidos em elementos filhos do elemento [&lt;body&gt;](body.md).

### Sintaxe { data-search-exclude }

| Elemento | Atributos | Conteúdo | Pai |
| --- | --- | --- | --- |
| [&lt;body&gt;](body.md) | *id* | ([&lt;port&gt;](port.md)  &#124; [&lt;property&gt;](property.md)  &#124; [&lt;media&gt;](media.md)  &#124; [&lt;context&gt;](context.md)  &#124; [&lt;switch&gt;](switch.md)  &#124; [&lt;link&gt;](link.md)  &#124; [&lt;meta&gt;](meta.md)  &#124; [&lt;metadata&gt;](metadata.md))* | [&lt;ncl&gt;](ncl.md) |

### Atributos { data-search-exclude }

| Nome | Valor | Descrição |
| --- | --- | --- |
| *id* | Pode receber qualquer valor de string que comece com uma letra ou sublinhado e que contenha apenas letras, dígitos, <"."> e <"_">. | O atributo *id* identifica exclusivamente o elemento dentro de um documento. |
