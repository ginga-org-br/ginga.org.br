---
title: Ginga-CC-WebServices
---

O Ginga-CC-WebServices é o subsistema de integração de Web Services do Ginga Common Core, habilitando chamadas de serviços web e serviços remotos no middleware do SBTVD (Sistema Brasileiro de Televisão Digital Terrestre).

Esta Norma estabelece um conjunto comum de WebServices suportado por implementações do núcleo do middleware Ginga (Ginga Common Core) em dispositivos receptores de televisão digital interativa. Ela define serviços para que os desenvolvedores de aplicações possam utilizar, bem como os requisitos necessários para os implementadores do middleware Ginga, garantindo a interoperabilidade das aplicações em diferentes implementações de plataformas de hardware.
