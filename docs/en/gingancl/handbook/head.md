### Definition and Usage { data-search-exclude }

The [&lt;head&gt;](head.md) tag is the container for all head elements.

Child elements of the [&lt;head&gt;](head.md) element promote reuse facilities in NCL
documents or define meta-information for NCL application user agents.

The [&lt;head&gt;](head.md) element does not have any attribute.
