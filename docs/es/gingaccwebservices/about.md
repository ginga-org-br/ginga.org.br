---
title: Ginga-CC-WebServices
---

Ginga-CC-WebServices es el subsistema de integración de servicios web de Ginga Common Core, habilitando invocaciones de servicios web y servicios remotos en el middleware del SBTVD (Sistema Brasileño de Televisión Digital Terrestre).

Esta Norma establece un conjunto común de WebServices soportado por las implementaciones del núcleo del middleware Ginga (Ginga Common Core) en dispositivos receptores de televisión digital interactiva. Define servicios para que los desarrolladores de aplicaciones puedan utilizar, así como los requisitos necesarios para los implementadores del middleware Ginga, garantizando la interoperabilidad de las aplicaciones en diferentes implementaciones de plataformas de hardware.
