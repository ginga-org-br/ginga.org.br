### Definición y uso { data-search-exclude }

El elemento [&lt;head&gt;](head.md) es el contenedor de todos los elementos principales.

Los elementos secundarios del elemento [&lt;head&gt;](head.md) promueven instalaciones de reutilización en NCL
documentos o definir metainformación para agentes de usuario de aplicaciones NCL.

El elemento [&lt;head&gt;](head.md) no tiene ningún atributo.
