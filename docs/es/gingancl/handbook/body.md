### Definición y uso { data-search-exclude }

El elemento [&lt;body&gt;](body.md) define el cuerpo del documento.

Todos los activos multimedia que definen una aplicación NCL junto con las relaciones.
entre estos activos se definen en elementos secundarios del elemento [&lt;body&gt;](body.md).

### Sintaxis { data-search-exclude }

| Elemento | Atributos | Contenido | Padre |
| --- | --- | --- | --- |
| [&lt;body&gt;](body.md) | *identificación* | ([&lt;port&gt;](port.md)  &#124; [&lt;property&gt;](property.md)  &#124; [&lt;media&gt;](media.md)  &#124; [&lt;context&gt;](context.md)  &#124; [&lt;switch&gt;](switch.md)  &#124; [&lt;link&gt;](link.md)  &#124; [&lt;meta&gt;](meta.md)  &#124; [&lt;metadata&gt;](metadata.md))* | [&lt;ncl&gt;](ncl.md) |

### Atributos { data-search-exclude }

| Nombre | Valor | Descripción |
| --- | --- | --- |
| *identificación* | Puede recibir cualquier valor de cadena que comience con una letra o un guión bajo y que solo contenga letras, dígitos, <"."> y <"_">. | El atributo *id* identifica de forma única el elemento dentro de un documento. |
