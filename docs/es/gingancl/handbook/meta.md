### Definición y uso { data-search-exclude }

El elemento [&lt;meta&gt;](meta.md) contiene metainformación sobre un documento NCL. eso
El reproductor NCL no lo utiliza para la presentación de la aplicación, pero puede
utilizado para otros agentes de usuario de NCL.

El elemento [&lt;meta&gt;](meta.md) especifica un único par propiedad-valor.

### Sintaxis { data-search-exclude }

| Elemento | Atributos | Contenido | Padre |
| --- | --- | --- | --- |
| [&lt;meta&gt;](meta.md) | <u>*nombre*</u>, <u>*contenido*</u> | -- | [&lt;body&gt;](body.md) o [&lt;context&gt;](context.md) |

### Atributos { data-search-exclude }

| Nombre | Valor | Descripción |
| --- | --- | --- |
| *nombre* | Cadena. | Nombre de metadatos. |
| *contenido* | Cadena. | Valor de los metadatos. |
