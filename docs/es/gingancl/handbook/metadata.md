### Definición y uso { data-search-exclude }

El elemento [&lt;metadata&gt;](metadata.md) contiene metainformación sobre una NCL
documento. El reproductor NCL no lo utiliza para la presentación de la aplicación,
pero se puede utilizar para otros agentes de usuario de NCL.

El elemento [&lt;metadata&gt;](metadata.md) actúa como elemento raíz de un árbol RDF.

### Sintaxis { data-search-exclude }

| Elemento | Atributos | Contenido | Padre |
| --- | --- | --- | --- |
| [&lt;metadata&gt;](metadata.md) | -- | -- | [&lt;body&gt;](body.md) o [&lt;context&gt;](context.md) |
