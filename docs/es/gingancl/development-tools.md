# Herramientas de Desarrollo

<div class="grid cards" markdown>

- :material-microsoft-visual-studio-code: **ginga-vscode**

    Extensión de VS Code para la ejecución de aplicaciones Ginga, basada en [gingaf](https://github.com/ginga-org-br/gingaf).

    [:octicons-arrow-right-24: Marketplace](https://marketplace.visualstudio.com/items?itemName=ginga-org-br.ginga-vscode)

- :material-play-circle: **ginga-playground**

    Entorno de ejecución en línea para experimentar con aplicaciones Ginga (NCL y HTML5), basado en [gingaf](https://github.com/ginga-org-br/gingaf).

    [:octicons-arrow-right-24: Probar](https://ginga-org-br.github.io/gingaf/ginga-playground)

</div>
