---
title: Sobre Ginga-J
---

Ginga-J es el entorno de ejecución procedural del middleware Ginga, basado en la plataforma Java. Permite la ejecución de aplicaciones de televisión interactiva codificadas en Java.

Esta Norma especifica los requisitos para la parte procedural del middleware para el Sistema Brasileño de Televisión Digital Terrestre (SBTVD). Garantiza que las aplicaciones puedan ejecutarse de forma consistente en diferentes implementaciones de plataformas de receptores.

Para la especificación técnica completa, consulte el documento [ABNT NBR 15606-4](/assets/pdf/NBR15606-4.pdf).
